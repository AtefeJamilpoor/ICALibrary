# ICALibrary
✓ قابلیت ثبت کاربر با مشخصات کامل (با کد کاربری یکتا) به همراه تصویر  
✓ قابلیت صدور کارت عضویت (با تصویر کاربری)  
✓ قابلیت ایجاد دسته بندی های مختلف برای کتب و جزوات و...  
✓ قابلیت ثبت کتاب، جزوه و ... با جزئیات کامل یک کتاب شامل ISBN و ایجاد بارکد برای هر کتاب بر اساس ISBN  
✓ قابلیت رزرو، تحویل و پس گرفتن کتاب  
✓ عدم ثبت درخواست جدید برای کاربرانی که دیرکرد در تحویل کتاب یا جزوه داشته اند ( با تنظیمات پیش فرض)  
✓ گزارش از لیست کاربران  
✓ گزارش از لیست کتب  
✓ گزارش گیری (HTML) برای دسته بندی کتاب ها به صورت Pivot Table  
✓ گزارش کتاب های رزرو شده بر اساس تاریخ انتخابی
